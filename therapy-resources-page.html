The one with all the podcast ads. Real therapists, weekly subscription model. Text, call, or video.

Cost $60-90/week
Time to Start 1-2 days
Visit BetterHelp →
Financial aid available. Can reduce cost by 40%.
Benefits: NYT on BetterHelp effectiveness Risks: WaPo on privacy issues

Talkspace

Similar to BetterHelp. Different pricing tiers based on what type of sessions you want.

Cost $69-109/week
Time to Start Same day
Visit Talkspace →
Benefits: NYT on Talkspace effectiveness Risks: WaPo on data privacy

Open Path Psychotherapy

Network of therapists offering lower rates. Pay once to join, then book sessions at reduced prices.

Cost $40-80/session
Time to Start 1-2 weeks
Join Open Path →
$65 lifetime membership if you make under $100k.
Benefits: NYT on affordable therapy networks Risks: WaPo on therapy network limitations

Community Mental Health Centers

Government-funded. They charge based on what you can afford. Long waits but legitimate option if you're broke.

Cost $0-40/session
Time to Start 2-4 weeks (varies)
Find Centers →
Benefits: NYT on community mental health centers Risks: WaPo on access challenges

University Training Clinics

Grad students need practice hours. You get cheap therapy. They're supervised by licensed psychologists.

Cost $15-50/session
Time to Start 2-3 weeks
Google "[your city] university counseling training clinic"
Benefits: NYT on university training clinics Risks: WaPo on training clinic limitations

Psychology Today Directory

Where therapists list their practices. Filter by sliding scale. You'll have to call/email a bunch.

Cost Varies widely
Time to Start 1-3 weeks
Search Therapists →
Filter by "sliding scale" and state your budget upfront.
Benefits: NYT on therapist directories Risks: WaPo on directory limitations

Employee Assistance Program (EAP)

Your job might offer free therapy sessions. Most people don't know this exists. Totally confidential.

Cost Free (limited sessions)
Time to Start 3-7 days
Ask HR: "Do we have an EAP?"
Benefits: NYT on EAP programs Risks: WaPo on EAP limitations

Cerebral

Therapy plus medication management. Quick to start but they push meds pretty hard.

Cost $85-325/month
Time to Start 1-3 days
Visit Cerebral →
Benefits: NYT on Cerebral accessibility Risks: WaPo on prescription practices

Sliding Scale Private Practice

Many therapists keep spots for people who can't afford full price. You have to ask though.

Cost $40-80/session
Time to Start 1-4 weeks
Say: "Do you have sliding scale spots? My budget is $X per session."
Benefits: NYT on sliding scale therapy Risks: WaPo on sliding scale limitations
