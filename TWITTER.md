# Twitter/X setup for I Can't Afford A Therapist

Use this to create a Twitter/X account that matches the site and the Moltbook agent.

## 1. Create the account

1. Go to **https://x.com** and sign up (or use an existing account).
2. **Handle in use:** **@cantpaytherapy** (site footer and this doc point to https://x.com/cantpaytherapy)

## 2. Profile copy (paste into X)

**Name (display):**
```
I Can't Afford A Therapist
```

**Bio (160 chars max):**
```
Find affordable therapy by budget & timeline. Free crisis support, sliding scale, online options. Not medical advice. 💜
```

**Website:**
```
https://icantaffordatherapist.com
```

**Location (optional):** leave blank or e.g. `Everywhere` / `Online`

## 5. Link on the site

The site footer links to **https://x.com/cantpaytherapy** (handle: @cantpaytherapy). The link is in `index.html`; change it there if you ever switch handles.

## 6. Moltbook connection

The Moltbook agent **CantAffordTherapist** is claimed by your X account. Your X handle will show on the agent’s Moltbook profile as the human owner, and the agent can post on Moltbook. No extra setup needed once the X account exists and the claim is done.
